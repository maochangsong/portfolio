Gameplay/UI Programming

Thanks to my experience studying at university and producing game projects, I have an understanding of almost every aspect of game programming.

Dialogue system & UI

I used Unity's TextMeshPro component to implement a dialogue system with its UI effects. It can output text in typewriter style and supports rich text effects, such as modifying the colour or font size of part of the text in a sentence. At the same time, I added annotation (Chinese or Japanese pronunciation symbols) function to it.
